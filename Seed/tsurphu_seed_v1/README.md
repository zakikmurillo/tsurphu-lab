# Tsurphu — Paquete Semilla v1 (YAML)

Este paquete contiene **diccionarios canónicos** y **políticas mínimas** para arrancar el motor Tsurphu con:

- Marco **Outer / Inner / Other** (Kalachakra)
- Taxonomía de **acciones** y **perfiles de día** (Key de Namkhai Norbu)
- **Relación elemental** persona↔día (mother/child/friend/enemy)
- Marco de **divinación elemental** (White Beryl) con la tortuga y sus 5 aspectos
- Catálogo semilla de **mandalas** (Mitrayogin / Mitro’s One Hundred) y compendios (Vajravali)
- Semillas para **longevidad / bla** (Gerke)

> Nota: Es un *seed* (semilla). No pretende estar “completo”; pretende ser **estable, auditable y expandible**.
